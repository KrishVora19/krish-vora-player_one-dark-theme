## Installation

### From VS Code Marketplace

1. Open VS Code  
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)  
3. Search for **Player One**  
4. Click **Install**

### From VSIX

1. Download the `.vsix` file  
2. Go to Extensions → ... → Install from VSIX  
3. Select your file
